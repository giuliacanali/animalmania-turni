Animalmania Turni V13.4 Extra Manuale + Pause

Modifiche:
- I dipendenti Extra non vengono mai generati automaticamente.
- Gli Extra restano visibili nei negozi principali e secondari assegnati.
- Gli Extra servono per correggere manualmente gli alert di scopertura.
- Nessun alert se Extra fa meno di 30h; alert solo se supera 30h.
- Alert di copertura base precisi: il negozio non può avere buchi durante l'apertura.
- Le pause lunghe 1h/2h/4h sui turni da 8h vengono gestite come presenza totale = 8h lavoro + pausa.
- Il motore genera un turno con pausa solo se entra davvero nell'orario del negozio.
